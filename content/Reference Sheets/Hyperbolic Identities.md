Instruction of hyperbolic functions in math classes in #precalc and calculus classes is #rare. Nonetheless, this reference page will prove to be useful should you require them. Let $x$, $\alpha$ (alpha), and $\beta$ (beta) be arbitrary values (see [[Greek letters]]).
# Basic Relationships

$$
\begin{align*}
\sinh x &= \frac{e^x - e^{-x}}{2} & \text{(hyperbolic sine)} \\[0.5em]
\cosh x &= \frac{e^x + e^{-x}}{2} & \text{(hyperbolic cosine)} \\[0.5em]
\tanh x &= \frac{\sinh x}{\cosh x} = \frac{1}{\coth x} &\text{(hyperbolic tangent)} \\[0.5em]
\coth x &= \frac{\cosh x}{\sinh x} = \frac{1}{\tanh x} &\text{(hyperbolic cotangent)} \\[0.5em]
\text{sech } x &= \frac{1}{\cosh x} &\text{(hyperbolic secant)} \\[0.5em]
\text{csch } x &= \frac{1}{\sinh x} &\text{(hyperbolic cosecant)}
\end{align*}
$$

# Pythagorean Identities
$$
\begin{gather*}
\cosh^2 x - \sinh^2 x = 1 \\
\tanh^2 x + \text{sech}^2 x = 1 \\
\text{coth}^2 x - \text{csch}^2 x = 1
\end{gather*}
$$

# Odd/Even Identities
$$
\begin{align*}
&\sinh(-x) &&= -\sinh x \\
&\cosh(-x) &&= \cosh x \\
&\tanh(-x) &&= -\tanh x \\
&\text{sech}(-x) &&= \text{sech } x \\
&\text{csch}(-x) &&= -\text{csch } x \\
&\text{coth}(-x) &&= -\text{coth } x
\end{align*}
$$
# Sum and Difference Formulas

$$
\begin{align*}
\sinh(\alpha \pm \beta) &= \sinh \alpha \cosh \beta \pm \cosh \alpha \sinh \beta \\
\cosh(\alpha \pm \beta) &= \cosh \alpha \cosh \beta \pm \sinh \alpha \sinh \beta \\[0.5em]
\tanh(\alpha \pm \beta) &= \frac{\tanh \alpha \pm \tanh \beta}{1 \pm \tanh \alpha \tanh \beta}
\end{align*}
$$
# Double-Value Formulas
$$
\begin{align*}
\sinh 2x &= 2 \sinh x \cosh x \\
\cosh 2x &= \cosh^2 x + \sinh^2 x \\
         &= 1 + 2 \sinh^2 \\
         &= 2\cosh^2 x - 1 \\[0.5em]
\tanh 2x &= \frac{2 \tanh x}{1 + \tanh^2 x}
\end{align*}
$$

# Half-Value Formulas
$$
\begin{gather*}
\sinh \frac{x}{2} = \pm \sqrt{ \frac{\cosh x - 1}{2} } \\[0.5em]
\cosh \frac{x}{2} = \pm \sqrt{ \frac{\cosh x + 1}{2} } \\[0.5em]
\tanh \frac{x}{2} = \pm \sqrt{ \frac{\cosh x - 1}{\cosh x + 1} }
\end{gather*}
$$
